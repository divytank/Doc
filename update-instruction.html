Complete Upload Guide

Step-by-step instructions for uploading files to your document manager

1
Prepare Your Files

Before uploading, make sure:

File naming: Use clear, descriptive names

Example: Project_Report_April_2024.pdf

File size: Keep files under 10MB for faster uploads

Large files can be compressed using tools like 7-Zip or WinRAR

File types supported:

PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, JPG, PNG, TXT, ZIP

Tip: Rename files before uploading to avoid confusion later.
2
Upload to Google Drive

Method A: Using Google Drive Website

1. Go to drive.google.com

2. Click "New" → "File upload"

3. Select your files

4. Wait for upload to complete

Method B: Drag & Drop

1. Open Google Drive

2. Drag files from your computer directly into the browser window

3. Drop files in the Google Drive folder

Method C: Google Drive Mobile App

1. Open Google Drive app

2. Tap "+" icon

3. Select "Upload"

4. Choose files from your phone

Success: Files are now uploaded to your Google Drive!
3
Get Download Link

Getting the Shareable Link

1. In Google Drive, right-click on your file

2. Select "Share" → "Share"

3. Click "Change to anyone with link"

4. Click "Copy link"

// Your link will look like this:
https://drive.google.com/file/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/view?usp=sharing

Convert to Direct Download Link

Change the link format from view to download:

// ORIGINAL (view link):
https://drive.google.com/file/d/FILE_ID/view?usp=sharing

// CONVERT TO (download link):
https://drive.google.com/uc?export=download&id=FILE_ID

Examples:

Before: https://drive.google.com/file/d/1AbCdEfGhIjKlMnOpQrStUvWxYz/view
After: https://drive.google.com/uc?export=download&id=1AbCdEfGhIjKlMnOpQrStUvWxYz
Quick Tip: Just replace /view with /uc?export=download&id= in the URL.
4
Add to Document Manager

For API Dashboard Users

1. Go to your dashboard

2. Click "Choose Files" or drag & drop

3. Files upload automatically to Google Drive

4. They appear instantly in your file list

For Simple Dashboard Users

1. In Simple Dashboard, find the "Add a File to List" section

2. Enter file name (e.g., report.pdf)

3. Paste the download link from Step 3

4. Click "Add to List"

Done! Your file is now available in your document manager.

Frequently Asked Questions

Why can't I upload directly from the website?

GitHub Pages is a static hosting service that doesn't allow file uploads. We use Google Drive as storage because it's free, secure, and provides 15GB of space.

My download link isn't working. What's wrong?

Make sure:

  1. The file is shared as "Anyone with link"
  2. You converted the link correctly (replace /view with /uc?export=download&id=)
  3. The file hasn't been deleted from Google Drive
Can I upload files larger than 10MB?

Google Drive allows files up to 5TB, but for better performance in the document manager, we recommend keeping files under 10MB. For larger files, consider compressing them or splitting into parts.

Is my data secure?

Yes! Your files are stored in your personal Google Drive account, protected by Google's security. The document manager only displays files you choose to share via links.

Can I access my files on mobile?

Absolutely! The document manager works on all devices. You can also use the Google Drive mobile app to upload files from your phone.

Pro Tips:

  • Create folders in Google Drive to organize your documents
  • Use consistent naming: YYYY-MM-DD_Description_Type.pdf
  • Bookmark your document manager for quick access
  • Regularly backup important files
